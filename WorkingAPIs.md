# Working API Endpoints

## User

### Register User
- **Endpoint:** `http://localhost:4000/api/user/register`
- **Method:** `POST`
- **Body:**
  ```json
  {
    "name": "Jon Snow",
    "password": "jonsnow"
  }

### Login User
- **Endpoint:** `http://localhost:4000/api/user/login`
- **Method:** `POST`
- **Body:**
  ```json
  {
    "name": "Jon Snow",
    "password": "jonsnow"
  }

### Delete User
- **Endpoint:** `http://localhost:4000/api/user/delete`
- **Method:** `DELETE`
- **Body:**
  ```json
  {
    "name": "Jon Snow",
  }
